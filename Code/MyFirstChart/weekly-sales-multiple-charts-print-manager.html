Using Print Manager

The chart below displays how to use Print Manager for Mozilla based browsers.
Print Manager prepares the chart for printing. PrintReadyStateChange event notifies when the chart is ready to print. This example shows an alert when the charts are ready for printing.
The 'Print Page' button helps to print the page along with the charts.
