The Details section lists the driver's recorded speeding events. By default, columns are sorted by descending order (most recent first).

 

Click an address in the Location column to see where an event occurred.

 

Click the Speed 55 icon to submit a Speed Limit Change Request.

 

If warranted, click Clear to delete a recorded event.

 

**NOTE**

Upon clearing a recorded event, the driver's overall speed score and their speed score for a particular posted speed limit range will not be immediately adjusted. The scores will be adjusted once the system performs a recalculation of such scores--generally, once an hour.

 

You can sort the events by ascending or descending order for any of the listed columns:

  • Location

  • Date/Time

  • Posted Speed

  • Avg Speed

  • Top Speed

  • Distance

You can also forgive any event by clicking Clear in the far right column.
