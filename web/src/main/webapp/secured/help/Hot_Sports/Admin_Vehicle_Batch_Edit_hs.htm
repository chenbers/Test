To batch edit several red flags at once, click the check box in the first column. Once you have selected the users, click the Batch Edit button. You will be able to edit:

  • Vehicle ID

  • Driver

  • Team

  • Year

  • Make

  • Model

  • Color

  • Unit

  • Type

  • VIN

  • Weight

  • License #

  • State

  • Status

Once you have made your changes, click Save. Your changes will become effective.
