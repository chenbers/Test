View Existing Organization

1.   Go to Admin page > Organization tab.

2.   View the current hierarchy of groups in the portal for your organization in the list displayed.

3.   To view details of a group displayed in the list, click on a group name to highlight it. The fields on the right refresh to display group details.

4.   To delete the highlighted group, click Delete button (upper-right).

5.   To edit the highlighted group, click Edit button (upper-right), make the changes needed, and then click Save. If the changes included a new name or new location in the organizationâ€™s hierarchy, the changes can be seen immediately in the refreshed hierarchy of groups.
