View All Trips

To view all trips for a driver or vehicle:

1.   Go to an individual Driver Performance or Vehicle Performance page (see "View Driver or Vehicle Performance Page" on page 49 ).

2.   Click view all trips above the upper-right corner of the Last Trip map.

 

 

 

 

 

Figure 51: View all trips

Driver Trips and Vehicle Trips pages initially display the last trip taken in the map.

To view a different trip, select the trip in the table. If the trip is not currently displayed in the table, use the navigation bar below the Trips table OR change the dates in the Data Range fields.

Trips table displays information regarding the last four trips, including the GPS status during the trip:

Good GPS

Bad GPS

Unknown GPS - Trip in progress
         or latent end of trip attribute

The inthinc® device tracks the GPS quality every second. At the end of the trip a GPS percentage notification is sent to the portal that indicates the percentage of time the GPS data is locked. The quality is calculated using the following formula:

goodPoints = gpsPercentage * tripDuration

goodTrip = (goodPoints / (tripDuration - 90 sec­onds)

If a goodTrip is greater than 90%, then the trip is considered good. Ninety seconds are subtracted to allow the GPS some time to acquire a lock. On trips less than 90 seconds the inthinc® device uses the gpsPercentage without any modifica­tions.

The navigation bar below the table allows you to scroll through previous trips within the specified range.

Data Range defaults to the last seven days. Use the calendar to select dates, then click Update Data Range to refresh the screen.

Stats section displays totals for trips taken within the data range.

Events section displays any driving notifications, such as speeding, hard brakes, hard accelerations, etc.

Settings section is used to set preferences.
