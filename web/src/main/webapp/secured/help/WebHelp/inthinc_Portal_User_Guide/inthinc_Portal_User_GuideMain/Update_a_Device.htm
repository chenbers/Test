Update a Device

Note: Devices are added to your system by an inthinc professional at the time of purchase. But you can update or edit those devices as needed to maintain correct associations with vehicles and users.

Update the device details in the system as needed.

1.   Go to Admin page > Devices tab.

Figure 31: Manage Device List

2.   Locate the device you want to update in the list, then click the edit link on the right side of the row. You may need to scroll to see the edit link. The Edit Device page opens with the Details tab selected (Figure 32)

Alternatively, to update two or more devices at the same time, select two or more check boxes (green highlight in Fig­ure 31) or the All box at the top of the column, then click Batch Edit. A Batch edit in inthinc(R) portal lets you edit select specific fields to update while retaining individual set­tings in other fields. For more information, see "Edit a Collection" on page 35.

Figure 32: Details tab on Edit Device page

3.   Make the changes necessary on the Details tab. For example, you may need to update the E-Call Phone number. Fields marked with an asterisk [*] are mandatory.

CAUTION!  The IMEI, SIM, and Device Phone numbers are specific to the tiwiPro™ device and should not be changed. Changes to these fields may cause the device to stop working.

For tiwiPro devices only:

a.   Under Device Information, update the fields as needed.

•   Device ID.

•   IMEI (International Mobile Equipment Identity number found on the tiwi™ label): This number is unique to every cell device and can be used to stop communication from tiwi™ in the event it is stolen.

•   SIM Card (A memory card inserted into your device by manufacturing).

•   Device Phone: This number is automatically assigned.

•   E-Call phone: This is the number that is called when you push the green button on your device.

b.   Under Device Profile, choose a status: New, Active, or Inactive.

c.   Under Device Assignment, click Assign. Select a vehicle for this device.

4.   Click Save.
