Add a Red Flag Notification

Red Flag Notifications are portal-user specific. The default owner is the username of the person who created the notifi­cation, but the Admin can change the owner name by updating the Alert Owner field now or at a later time.

Note: Custom Red Flags will not affect In-Cab mentoring or Driver Performance scores. To set up In-Cab mentoring, edit Vehicle or Device page.

 

1.   Go to Admin page > Red Flags tab, then click Add Red Flag. The Add Red Flag form displays.  


 Figure 37: Add Red Flag for Emergency


2.   In the Red Flag Type field, choose a Red Flag Type: Speeding, Aggressive Driving, Compliance, Crash, Tampering, Emergency, Zones, Vehicle, HOS, Text Message, Wireline Alarm, Fatigue, Installation, or Off Hour Driving.


3.   In the Name, Description, & Type section:

 
•   Name: Name the red flag.

 
•   Description: Description the red flag (optional).

 
•   Notification: Choose the Notification type for this red flag: Information, Warning, or Critical. 


 

  

   

   
   
Note: To specify how a user is notified for each Notification type (Information, Warning, or Critical), edit the user’s page (see "Update a User" on page 27). There you can choose whether the user receives an email, text message, a phone call, or no notification at all.




  


 

 

 
•   Status: Select a status: Active or Inactive.

 
•   Which Days and Time Frame: Select the days of the week that you want to be alerted and the time frame. For example, you might not want to be alerted on Saturdays and Sundays, so you would not select those days.


4.   In the Assigned To section:

 
•   Build your own Assign To list using the arrows and list of teams provided.


 

  

   

   
   
Note: Employees must be listed as drivers on the inthinc® Portal portal to receive notifications.




  


 

 

 
•   Select an Alert Owner from the drop-down list (default is the person creating the alert). This is the person who will be able to see and modify the alert.


5.   If the red flag type you selected has a type-specific preference area (such as the red-highlighted Zone preferences addition shown in Figure 38), customize the red flag as needed.


6.   In the Notifications By Name area: Enter a name or choose one from the list of existing names. The notification will be sent using the method assigned in the Information field in the user’s settings.


7.   In the Notifications By E-mail area: For first-level escalation, enter one to three email addresses.


8.   In the Notifications By Phone area: For second-level escalation, enter one to three names of people to be called.


9.   Once you have set your red flag preferences, click Save; OR to dismiss changes and start over, click Cancel.
