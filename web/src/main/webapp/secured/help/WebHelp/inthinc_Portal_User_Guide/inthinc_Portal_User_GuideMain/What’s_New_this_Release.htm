What’s New this Release

The inthinc® web portals for fleets, tiwiPro™ and GAIN™, were merged. The new portal name is inthinc® Portal. One portal now supports all vehicles in a fleet that use either a tiwiPro™ or waySmart™ device. This document was based on the tiwiPro™ Portal User Guide and was then enhanced to document the waySmart™ features incorporated into the soft­ware. While this is technically a new document, tiwiPro™ users may find this “What’s New...” section helpful.
