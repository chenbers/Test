Update a Vehicle

1.   Go to Admin page > Vehicles tab.
2.   Locate the vehicle you want to update in the list, then click Edit (in the right margin).
3.   Make the necessary changes to the Edit Vehicle form.

•   Details tab: Fields marked with an asterisk (*) are mandatory.

•   Speed & Sensitivity tab: Set preferences here or on the associated Edit Device form to set up in-cab mentoring of driver. For details about these features, see "Set Up In-Cab Mentoring" on page 32.

Note: Changes to Sensitivity Levels do not take effect until the vehicle is powered on again. Changes are then reflected on your inthinc® Portal account profile.

 

4.   When finished editing, click Save.
