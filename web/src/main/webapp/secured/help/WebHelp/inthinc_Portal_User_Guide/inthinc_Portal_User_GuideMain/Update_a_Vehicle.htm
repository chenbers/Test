Update a Vehicle

1.   Go to Admin page > Vehicles tab.

2.   Locate the vehicle you want to update in the list, then click Edit (in the right margin).

Alternatively, select one or more vehicles boxes or check All box (top of column), then click Batch Edit.

A Batch edit in tiwiPro™ Portal lets you edit two or more objects in a list or the entire list. After selecting two or more objects (such as Vehicles in Figure 23) and clicking Batch Edit, you can select specific fields to update for the collection while retaining individual settings in other fields. For more information, see "Edit a Collection" on page 35.

3.   Make the necessary changes to the Edit Vehicle form.

•   Details tab: Fields marked with an asterisk (*) are mandatory.

•   Speed & Sensitivity tab: Set preferences here or on the associated Edit Device form to set up in-cab mentoring of driver. For details about these features, see "Set Up In-Cab Mentoring" on page 38.

Note: Changes to Sensitivity Levels do not take effect until the vehicle is powered on again. Changes are then reflected on your tiwiPro™ Portal account profile.

4.   When finished editing, click Save.
