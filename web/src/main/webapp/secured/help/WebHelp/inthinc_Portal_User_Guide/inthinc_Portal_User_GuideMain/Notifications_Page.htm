Notifications Page

When you choose Notifications in the Navigation Bar, the Red Flags page (Figure 16) displays by default.

Types of Notifications: You can view Red Flags, Safety, Diagnostics, Zones, HOS Exceptions, Emergency, and Crash His­tory notifications by clicking the respective option just below the main navigation bar. These notifications (1) provide group summaries in spreadsheet format and (2) provide access to individual notifications sent regarding drivers or vehi­cles.

Sort Options: Reports can be sorted by a variety of columns. Arrows in column headings indicate additional sort options: Clicking any of the column headings once will reverse the sort order to either an ascending or descending order.

Note: To customize columns, see "Customize Columns in Reports and Notifications" on page 48.
