Notifications Page

When you choose Notifications in the Navigation Bar, the Red Flags page (Figure 14) displays by default.

Types of Notifications: You can view Red Flags, Safety, Diagnostics, Zones, HOS, Emergency, and Crash History notifica­tions by clicking the respective option just below the main navigation bar. These notifications (1) provide group summa­ries in spreadsheet format and (2) provide access to individual notifications sent regarding drivers or vehicles.

Sort Options: Reports can be sorted by a variety of columns. Arrows in column headings indicate additional sort options: Clicking any of the column headings once will reverse the sort order to either an ascending or descending order.
