Set Up Mentoring, Zones, and Notifications

You can set mentoring, zones, red flag notifications, and report delivery preferences from the Admin page.

•   View or Edit Admin Portal Account allows user logged in as Admin to: maintain company-specific displays in the portal, update web map service URL (location where system can access your company maps to be overlaid on Google Maps), and maintain Portal Support and Admin contact information.

•   Set Up In-Cab Mentoring (also known as Vehicle/Device Speed & Sensitivity settings), including driver behavior sensitivities (speed, aggressive behaviors, seat belt usage, etc.).

•   Create a Zone (related to geographic locations).

•   Add a Red Flag Notification (to determine how and when you will be notified of driving events). Decide when and how you will receive alerts from inthinc® Portal.

•   Submit Speed-Limit Change Request provides an easy way to notify us if our speed limit data needs to be updated.
