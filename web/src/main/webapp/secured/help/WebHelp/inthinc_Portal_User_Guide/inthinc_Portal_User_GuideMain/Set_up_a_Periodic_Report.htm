Set up a Periodic Report

Use the procedures listed in this section to set up and deliver routine reports for all of your inthinc® in-vehicle devices.

Note: For waySmart™ device users who want to generate a one-time report, see the alternate report­ing methods described in "Use waySmart™ Messaging and Reporting Features" on page 44.

The Reports tab in the Admin page lets you specify which reports are E-mailed to you on a periodic basis. Reports avail­able are:

•   Fleet/Division reports show scoring breakdown, trend, and fuel efficiency information similar to the Home page. This report lets you select the division or team and the time frame of the report.

•   Driver and Vehicle reports let you select the division or team, and the time frame of the report.

•   Idling and Devices reports let you select the division or team of the report.

•   waySmart™ reports (available for waySmart™ users) let you select one of many waySmart™ reports.

•   Team Statistics report contains the same information as the Team Driver Statistics page. This report lets you select the team and the time frame of the report.

Note: The list of available reports varies based on the inthinc in-vehicle devices being used.

To add a report to your list of scheduled reports: 

1.   Go to Admin page > Report tab, then click Add Report. The Add Report page displays (Figure 48).
2.   Complete the Add Report form to schedule the delivery of a report, then click Save.
