View HOS Log for a Driver

waySmart™ device users who also have Hours of Service (HOS) features enabled can view a driver’s HOS log.

Note: To schedule this report for periodic delivery, see "Set up a Periodic Report" on page 43.

 

1.   After logging in to the portal, go to the HOS page. The Driver Log page is selected by default.
2.   Choose a driver from the drop down list (Figure 53).
3.   Click the calendar icons to set the date range dates.
4.   Click Refresh to view the log.
