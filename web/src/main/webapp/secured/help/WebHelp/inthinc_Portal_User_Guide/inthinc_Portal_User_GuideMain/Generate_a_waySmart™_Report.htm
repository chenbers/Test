Generate a waySmart™ Report

Users of the waySmart™ device can use these instructions to generate and view a waySmart™ report.

Note: To schedule this report for periodic delivery, see "Set up a Periodic Report" on page 43.

 

1.   In the portal, go to Reports > waySmart.
2.   Choose a report type from the drop down list (see Figure 52).

The waySmart™ report options are listed in Table 2.

3.   Set filters for the report. Filter options vary per report. Two common filters are Data Range and Group(s):
a.      Click the calendar icons to set the date range.
b.   Use the Group(s) drop-down list to select one or more groups to include. 
3.   To generate the report, choose a Report delivery method. Options vary per report, but may include: Table, HTML, PDF, Excel, or E-mail. If you choose E-mail, enter an E-mail address in the dialog box that displays. All other reports display immediately in the current page or a separate window.

4.   View the Report:

 
•   Table - Displays in portal table format (with sortable columns) in the bottom of the current page.

 
•   HTML - Displays in HTML format in the bottom of the current portal page.

 
•   PDF -Displays in a PDF Reader application window.

 
•   Excel - Displays in an Excel application window.

 
•   E-mail - Report is delivered to the Email address provided.
