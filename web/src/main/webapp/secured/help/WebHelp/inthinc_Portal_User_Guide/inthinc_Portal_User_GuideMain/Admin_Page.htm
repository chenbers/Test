Admin Page

When you choose Admin in the Navigation Bar, the Admin page (Figure 18) allows you to develop and maintain your fleet. The tabs on the left provide access to system components.

Note: The Admin tab is only viewable if your login account has been granted Custom User Rights. If you need these rights and don’t currently have them, request that your inthinc® Portal administrator add a role with more access rights to your user account.

 

 Figure 18: Admin Page

System Administration tasks that can be completed from this page include:

•   Users: Add, update, or delete a user

•   Vehicles: Add, update, or delete a vehicle

•   Devices: Update or delete a device

•   Zones: Create, edit, or delete a zone

•   Red Flags: Set up a Zone alert or Red Flag notification

•   Reports: Schedule the delivery of a report

•   Organization: Add divisions or teams in a hierarchy

•   Custom Roles: Create a custom role (used for defining web access rights)

•   Speed Limit: Request a speed limit correction

•   Account: View or edit portal account information for a selected user, or configure your company’s custom spatial map over the Google map.
