Admin Page

When you choose Admin in the Navigation Bar, the Admin page (Figure 20) allows you to develop and maintain your fleet. The tabs on the left provide access to system components.

Note: The Admin tab is only viewable if your login account has been granted Custom User Rights. If you need these rights and don’t currently have them, request that your tiwiPro™ Portal administrator add a role with more access rights to your user account.

As with most tables in the portal, you can change the way the column is sorted by clicking the column heading. You can also select which columns are displayed by clicking the Edit Columns link. For more information, see "Customize Col­umns in Reports and Notifications" on page 48.

A Batch edit in tiwiPro™ Portal lets you edit two or more objects in a list or the entire list. After selecting two or more objects (such as Users in Figure 20) and clicking Batch Edit, you can select specific fields to update for the collection while retaining individual settings in other fields. For more information, see "Edit a Collection" on page 35.

 

Figure 20: Admin Page

System Administration tasks that can be completed from this page include:

•   Users: Add, update, or delete a user

•   Vehicles: Add, update, or delete a vehicle

•   Devices: Update or delete a device

•   Zones: Create, edit, or delete a zone; copy zone files to local computer

•   Red Flags: Set up a Zone alert or Red Flag notification

•   Reports: Schedule the delivery of a report

•   Organization: Add divisions or teams in a hierarchy

•   Custom Roles: Create a custom role (used for defining web access rights)

•   Speed By Street: Request a speed limit correction

•   Account: View or edit portal account information for a selected user, or configure your company’s custom spatial map over the Google map.
