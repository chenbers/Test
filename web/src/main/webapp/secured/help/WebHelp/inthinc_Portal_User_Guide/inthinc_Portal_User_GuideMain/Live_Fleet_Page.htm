Live Fleet Page

When you choose Live Fleet in the Navigation Bar, the Live Fleet page (Figure 17) provides a global view of the current location of your entire fleet, division or team, depending on your level of access.

A refresh button (see green highlight) is provided to force the Live Fleet display to refresh immediately.

Clicking the driver name in the list takes you to the Driver Performance page for that driver.

Clicking the vehicle name in the list takes you to the Vehicle Performance page for that vehicle.

Clicking the vehicle icon in the list will locate that vehicle in the map. This makes it easy to locate a specific vehicle.

To find vehicles nearest a specific location, enter an address in the Find Address field, then click Locate (see red highlight).
