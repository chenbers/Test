Appendix B -    Notification Samples

This section includes sample notifications available to tiwiPro™ Portal Users. Some notifications are available to both tiwi™ device users and waySmart™ device users, while others are only available to waySmart™ device users or waySmart™ device users with HOS features enabled. Any waySmart™ exception is indicated for each report.

The Alert column these delivery state links: No Alert, Sent, Pending, or Canceled.

•   No Alert — No alert was sent (based on preferences set).

•   Sent — A notification was sent and acknowledge or completed the escalation process.

•   Pending — A notification was sent and an escalation is in process.

•   Canceled — The alert was canceled before delivery was completed.

When you click a link in the Alert column, an Alert Details page displays where you can select the specific notification.
