Export or Email a Crash Report

1.   Click the export utility (wrench) icon on the far right of the row that contains the Search button.

Figure 63: Utility Icon for Exporting Reports

The portal displays three options:

•   E-mail This Report

•   Export to PDF

•   Export to Excel

2.   If you select Email This Report, the portal will display a dialog allowing you to supply email addresses, separated by a comma, to where you would like to send the report. If you click Cancel, the report is not sent and you are returned to where they were in the portal prior to selecting the option. If you select E-mail, the report is sent to the email addresses in the list, and you are returned to where you were in the portal prior to selecting the option.

3.   If you select Export to PDF, an Opening tiwiPro_Report.pdf dialog opens. allowing you to select from a set of radio buttons.

•   Open with....

•   Save File

•   OK

•   Cancel

If you click OK, a PDF report is generated and saved.

•   If you are using Firefox, it saves to your pre-determined location.

•   If you are using Internet Explorer, a File Download dialog opens and prompts you to either open or save the document. If you click Save, a Save As dialog opens. Select the desired location to where you would like to save the file, then click Save.

•   If you click Cancel, the report is not generated, and you are returned to where you were in the portal prior to selecting the option.

4.   If you select Export to Excel, the portal will display a dialog allowing you to select from a set of radio buttons:

•   Open with....

•   Save File

•   OK

•   Cancel

If you click OK, a spreadsheet report is generated and saved.

•   If you are using Firefox, it saves to your pre-determined location.

•   If you are using Internet Explorer, a File Download dialog opens and prompts you to either open or save the document. If you click Save, a Save As dialog opens. Select the desired location to where you would like to save the file, then click Save.

•   If you click Cancel, the report is not generated, and you are returned to where you were in the portal prior to selecting the option.
