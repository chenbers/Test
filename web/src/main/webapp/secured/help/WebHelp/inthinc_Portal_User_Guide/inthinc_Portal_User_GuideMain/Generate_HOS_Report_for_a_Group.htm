Generate HOS Report for a Group

Users of the waySmart™ device who also have Hours of Service (HOS) features enabled can use these instructions to gen­erate and view an HOS group report.

Note: To schedule this report for periodic delivery, see "Set up a Periodic Report" on page 43.

 

1.   After logging in to the portal, go to the HOS page.
2.   Choose a report type from the drop down list (Figure 54).
3.   Set filters for the report. Filter options vary per report. Two common filters are Data Range and Group(s):
a.      Click the calendar icons to set the date range.
b.   Use the Group(s) drop-down list to select one or more groups to include. 
3.   Choose a Report delivery option. Options vary per report, but may include: Table, HTML, PDF, Excel, or E-mail. If you choose E-mail, enter an E-mail address in the dialog box that displays. All other reports display immediately in the current page or a separate window.

The HOS report options are listed in Table 3.


 

  

   

   
   
Table 3:  HOS Reports 



  

 

 

  

   

   
   
Report


   
   
Description




   

   
   
HOS Daily Driver Log Report


   
   
Report showing a graphical representation of a selected driver’s HOS log for each day during the selected time period.




   

   
   
HOS Violation Summary Report


   
   
The HOS Violations summary provides two different types of data - rule viola­tions and mileage violations. Additionally, this report displays the number of drivers for the selected group, and the total miles (on and off road).




   

   
   
HOS Violation Detail Report


   
   
(For driver or group) A report listing drivers, time, type, and duration of HOS violations in the selected group within the specified date range.




   

   
   
HOS Driver DOT Log Report


   
   
Report showing time, date, vehicle, and other information for each HOS sta­tus change for the scheduled driver within the specified time period. The report includes all changes/edits and the user who made them.




   

   
   
DOT Time Remaining


   
   
(List of Driver Hours for Group) The number of hours a Driver has remaining to drive today (“Hrs Remain Today” column) based on the DOT rules. Also shows the on-duty-driving and off-duty-not-driving hours that contribute to the Hours Remaining Today calculation for the last 14 days.




   

   
   
HOS Zero Miles


   
   
Zero miles are counted when a vehicle is moving but no driver is logged in. Once a group and a date range have been specified, the zero miles by vehicle report gives the total number of zero miles listed by vehicle.




   

   
   
HOS Edits


   
   
This report will provide information on who is editing HOS logs. Use this report to audit changes to HOS logs.
