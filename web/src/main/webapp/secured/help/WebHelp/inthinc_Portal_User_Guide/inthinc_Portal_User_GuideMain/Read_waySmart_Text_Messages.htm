Read waySmart Text Messages

Use these instructions to read text messages sent from your waySmart™ drivers, vehicles, or group members.

1.   In the toolbar at the top of the page, click My Messages. The Inbox tab is selected by default (see Figure 55).

2.   (Optional) Set filters to narrow the list:

a.   Choose a team from the drop-down list.

b.   Set Date Range using the calendar icons for the Start and End dates.

c.   Click Apply Filter.

4.   If necessary, use the scroll buttons at the bottom of the screen to see all messages in your Inbox.

5.   Double-click any message in the list to display it. You may need to use the scroll buttons at the bottom of the screen to see all messages in your Inbox.

6.   To reply to one or more messages at a time:

a.   Click each applicable check box to select, then click the “Reply to selected” link. The Compose page displays.

b.   Complete the Compose Message page (described in "Compose a waySmart Text Message" on page 54).
