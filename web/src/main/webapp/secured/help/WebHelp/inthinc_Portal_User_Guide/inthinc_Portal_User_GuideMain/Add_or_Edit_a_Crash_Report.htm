Add or Edit a Crash Report

To add/edit a report, preform the following:

1.   Log in to the portal inthinc® Portal portal at http://www.stage.inthinc.com.
2.   From the Home page menu, select Notification > Crash History. The Crash History page is displayed. 
3.   Click Add Crash Report to create a new report, or...
Click Details to display the Crash Report, then click Edit. 
4.   If you opt to create a report from an existing trip, select a driver or vehicle first, then select the trip to which you would like to assign the report. You can only select a point that is located within the trip’s given points. 
5.   If you select to create a report from scratch, you are given the option to select any point on a map. 
6.   If you select a point from an existing trip, some of the crash reports additional information will be auto filled based on the available information from the trip. 
7.   Fill in the following fields:

•   Status: New/Confirmed/Excluded

•   Date/Time

•   Location: Address (map and text field). Text field is automatically completed when location point is selected in the map. The user can override this data and enter in their own data.

•   Weather

•   Vehicle

•   Driver

•   Description

•   Detailed Crash Data (Read only, includes list of Crash Data Points sent by the device when a crash is detected)

•   Occupant Count

Note: While most of the fields can be edited, the following fields are read-only fields:

•   Date/Time (Read only if data exists in the note and the data in the note is NOT an estimate. There should be data that indicates if the Date/Time was an estimate)

•   Location: Address (Read only if data exists in the note. This will come from the detailed crash data and may be a series of points)

 

8.   After filling in the required crash report data, click Save. The crash report will be saved to the database.
