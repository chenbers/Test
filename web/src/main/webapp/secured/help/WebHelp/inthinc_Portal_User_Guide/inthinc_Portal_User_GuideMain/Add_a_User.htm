Add a User

A user can be added to create a tiwiPro™ Portal login account (with vari­ous levels of access rights) or to iden­tify a driver in the system, or both.

When you specify all of the roles that apply to this user, you are setting portal access rights for the user based on the assigned group and those roles.

Tip: Create custom roles (or have your Admin create custom roles) for your organization before adding users to avoid having to edit user objects later.

Note: A user in the system is assigned one Group, but can be assigned multi­ple roles. The role assigned to a user applies throughout the group that they are assigned. So the higher the assigned group in the company, the more authority the user has.

When you indicate that a user is a driver, this user can be associated with a vehicle or an inthinc® device. Each time you add a driver, you can specify the types of Alerts the system sends you regarding the performance of that driver. For example, you may only want to be notified of speeding events only of an extreme nature, such as going 20 miles over the speed limit.

 

Figure 27: Add User form

To add a new user:

1.   Click Admin on the navigation bar. The Admin page opens with the User tab selected by default.

2.   Click Add User (on the left). The Add User form displays.

3.   Complete the Add User form. The following fields are required (as indicated by asterisks [*] on the form):

•   User Information: First Name, and Last Name

•   Driver Information (when checked): Team (choose one from list) and Status (choose one from list)

•   RFID Information (RFID Card user only): RFID Bar Code (number below bar code)

•   Employee Information: Time Zone (choose one from list)

•   Login Information (when checked):

•   User Name

•   Password

•   Password Again

•   Group (choose one from list)

•   Role (choose one from list)

•   Status (choose one from list)

•   Notifications: E-mail 1 (e-mail address of user you are creating) — This is required only if the Login Informa­tion box is checked.

Note: In order for tiwiPro™ Portal to track and report driver information, you must assign a user to the vehicle.

Note: Create as many custom roles as needed to meet your login access needs. See "Add a User" on page 30.

Note: If you use the RFID login option, you must associate the RFID card with the driver. For instruc­tions to do so, see "Set Up RFID Card for a Driver of tiwi Vehicle" on page 34.

4.   Click Save to save your changes and add the new user to the system OR click Cancel to exit without saving.
