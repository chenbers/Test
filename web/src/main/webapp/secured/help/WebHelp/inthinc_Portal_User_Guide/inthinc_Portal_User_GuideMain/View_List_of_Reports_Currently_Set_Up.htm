View List of Reports Currently Set Up

Figure 44: View list of Reports set

 

1.   Go to Admin page > Reports tab.

2.   View the list of Reports currently set up or defined for your organization.

3.   To view the current setup or definition of a report in the list, click the report name.

4.   To edit this report, click the Edit button (upper-right of page), edit the fields, and then click Save.
