What is New this Release

The inthinc® web portals for fleets, tiwiPro™ and GAIN™, were merged. The new portal name is inthinc® Portal. One portal now supports all vehicles in a fleet that use either a tiwiPro™ or waySmart™ device. This document, inthinc® Portal User Guide, was based on the tiwiPro™ Portal User Guide and was then enhanced to include the waySmart™ fea­tures incorporated into the software. While this is technically a new document, tiwiPro™ device users may find this “What is New...” section helpful.
