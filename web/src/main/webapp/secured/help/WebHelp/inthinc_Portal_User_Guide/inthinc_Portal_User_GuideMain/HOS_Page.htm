HOS Page

The HOS option in the Navigation Bar is available only for waySmart™ users who also have the Hours of Service (HOS) feature enabled. The HOS link provides access to two types of Hours of Service pages: HOS Driver Logs and HOS Reports.

•   HOS Driver Logs (Figure 18) provides a list of specific time per status for the selected driver.

On the Driver Logs page, you can Add, Edit, or Batch Edit logs (see green highlights).

•   Add or Edit displays a form to add or edit for a single log.

•   Batch Edit allows you to edit select fields in two or more logs at a time; you ca edit select fields while retaining individual settings in other fields. For more information, see "Edit a Collection" on page 35.

•   HOS Reports (Figure 19) provides access to a long list of HOS-specific reports that can be viewed.
