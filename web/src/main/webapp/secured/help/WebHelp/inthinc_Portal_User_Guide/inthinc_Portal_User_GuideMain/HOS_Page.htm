HOS Page

The HOS option in the Navigation Bar is available only for waySmart™ users who also have the Hours of Service (HOS) feature enabled. The HOS link provides access to two types of Hours of Service pages: HOS Driver Logs and HOS Reports.

•   HOS Driver Logs (Figure 16) provides a list of specific time per status for the selected driver.

•   HOS Reports (Figure 17) provides access to a long list of HOS-specific reports that can be viewed.
