Set Up RFID Card for a Driver of tiwi™ Vehicle

1.   Log in to the inthinc® Portal as Admin or as a user with rights to edit User pages.
2.   Go to the individual user's page.
3.   Locate the bar code number of the RFID Card you want this user to use: The bar code number is located UNDER the bar code on the back of the card. 
4.   Enter the bar code number in the Bar Code field and then click SAVE. Once the device is powered down and powered back up again, the user should be able to log in to the tiwi™ using the RFID Card (as described in the tiwi™ for fleets Driver User Guide).
