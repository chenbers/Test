Review waySmart™ Sent Messages

Use these instructions to review text messages you have sent to your waySmart™ drivers, vehicles, or groups. 

1.   In the toolbar at the top of the page, click My Messages. Your Inbox tab is selected by default.
2.   Click Sent tab (on the right). The Sent Items page displays (see Figure 51).
3.   (Optional) Set filters to narrow the list:
a.   Choose a team from the drop-down list.
b.   Set Date Range using the calendar icons for the Start and End dates.
c.   Click Apply Filter.
4.   If necessary, use the scroll buttons at the bottom of the screen to see all messages in your Sent box.
5.   Double-click any message in the list to display it.
