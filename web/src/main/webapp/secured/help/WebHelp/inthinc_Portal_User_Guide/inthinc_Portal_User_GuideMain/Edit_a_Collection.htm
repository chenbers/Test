Edit a Collection

Batch editing is efficiently updating any collection of portal objects (such as drivers, devices, vehicles, or logs) that are available for individual updates.

A Batch edit in inthinc Portal lets you edit selected fields in two or more objects in a list or in the entire list. After selecting the objects and clicking Batch Edit, you can select specific fields to update for the collection while retaining individual settings in other fields.

For example. If you need to batch edit a collection of users:

1.   Go to Admin page > Users.

2.   In the list, select one or more individual rows containing the User you want to update (Figure 35, red highlight).

Alternatively, you can select the All box at the top of the column (Figure 35, black highlight).

3.   Click Batch Edit.

4.   In the form displayed, select boxes next to the fields you want to update (Figure 36, green highlight).

5.   In the selected field, change the settings and then click Save (Figure 36, red highlight).

Only the fields you selected will be updated and then only in the users that you selected.
