Crash History Notifications

Red Flag notifications can be listed for a selected team for a selected time frame.  

Filters: List can be filtered by team.

Sort Options: List can be sorted alphanumerically by Date/Time, Group, Driver, Vehicle ID, Occupants, Status, or Weather.

 

To view a list of Red Flag notifications:

1.   Go to Notifications > Crash History.
2.   Choose a Team and then click Refresh.
3.   (Optional) Use a sort option: Date/Time, Group, Driver, Vehicle ID, Occupants, Status, or Weather. The list of Crash Histories is displayed as requested.
