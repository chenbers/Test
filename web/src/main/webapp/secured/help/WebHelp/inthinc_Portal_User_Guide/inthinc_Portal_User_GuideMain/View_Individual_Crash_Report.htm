View Individual Crash Report

Crash Data is displayed on the inthinc® Portal por­tal on a Crash Report page.

Note: You must have Java installed to view the Crash Trace data in this report as designed. If you don’t have Java installed, an error message will display. Follow these steps to install Java:

1.   In the message, click www.java.com link.
2.   Click Free Java Download button.
3.   Click Agree and Start Free Download.
4.   Click Run.
5.   At “Welcome to Java” screen, click Install.
6.   At the message “You have successfully installed Java,” click Close. You are returned to your previous portal page. Refresh the screen before proceeding.

To locate the crash report for a specific crash:

1.   Log in to the inthinc® Portal portal.
2.   Go to Notifications page > Crash History page.
3.   Choose a Team and then click Refresh.
4.   In the list, locate the crash data report you would like to view (or use Search feature), then click the Details link in the far right column.

The crash report displays with crash-specific and vehicle-specific information (see Figure 56). The Crash Report includes the following:

•   Crash Details:

•   Status: New, Confirmed, or Excluded

•   Date/Time

•   Vehicle Name, Driver Name, Weather, Occupant Count, Description

•   Crash Route: Map with route overlay

•   Crash Events: Trip & Speed/RPM Graph: Includes 2.5 minutes of trip data (updated every 15 seconds) and Driver Seat belt.

•   Crash Trace: Includes 30 seconds of trip data, with 20 seconds pre-crash and 10 sec­onds post-crash event.

Crash Trace section has three tab:

•   Linear Acceleration (the default view)

•   Linear Velocity

•   Visualization

5.   On crash reports that have more than 25 details listed, click the arrows on the Navigation Bar to navigate to the next, previous, first or last page. Additionally, you can click on the desired number to display the appropriate page.
