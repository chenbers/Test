Update a User

1.   Go to Admin page. The Users page with the list of users is displayed by default.
2.   In the Name column, locate the name of the user you need to edit, then click the edit link for that name (see Fig­ure 25).


 Figure 25: Edit User


3.   Update the Edit User form as needed, then click Save.


 

  

   

   
   
Note: If you move a driver from one team to another, that driver’s history will stay with the first team and remain calculated into that team’s score. Any new driving history will affect the new team’s score.
