Update a User

1.   Go to Admin page. The Users page with the list of users is displayed by default.

2.   In the Name column, locate the name of the user you need to edit, then click the edit link for that name (see red highlight in Figure 28).

Alternatively, select one or more users or check All box (black highlight), then click Batch Edit.

A Batch edit in tiwiPro™ Portal lets you edit two or more objects in a list or the entire list. After selecting two or more objects (such as Users in Figure 28) and clicking Batch Edit, you can select specific fields to update for the col­lection while retaining individual settings in other fields. For more information, see "Edit a Collection" on page 35.

3.   Update the Edit User form as needed, then click Save.

Note: If you move a driver from one team to another, that driver’s history will stay with the first team and remain calculated into that team’s score. Any new driving history will affect the new team’s score.
