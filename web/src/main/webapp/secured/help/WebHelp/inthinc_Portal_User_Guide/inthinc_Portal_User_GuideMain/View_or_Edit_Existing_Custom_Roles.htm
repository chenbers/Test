View or Edit Existing Custom Roles

1.   Go to Admin page > Custom Roles tab. The list of existing custom roles for your organization is displayed.

2.   To view details of a custom role, click the name of the custom role. When finished viewing, click “Back to Custom Roles” link at the top.

3.   To edit the custom role, click Edit link (link on right side of row), make the changes needed, and then click Save. If the changes included a new name, the changes can be seen immediately in the refreshed list.

4.   To delete a custom role, select the custom role (check the box or click on the name), then click Delete.
