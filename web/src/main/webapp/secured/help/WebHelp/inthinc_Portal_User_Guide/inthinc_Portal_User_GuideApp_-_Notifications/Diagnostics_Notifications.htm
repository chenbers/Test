Diagnostics Notifications

Diagnostics notifications can be listed for a selected team for a selected time frame.  

Filters: List can be filtered by Team, Time Frame, and/or Category. The Category options are displayed in Figure 7.

Sort Options: List can be sorted alphanumerically by Group, Driver, or Vehicle ID.

 

To view a list of Diagnostics notifications:

1.   Go to Notifications > Diagnostics.

2.   Choose a Team and a Time Frame, and then click Refresh.

3.   (Optional) Further filter the list by selecting a Category. The list of available categories is displayed in Figure 7.

4.   (Optional) Use a sort option: Date/Time, Group, Driver, or Vehicle ID. The list of notifications is displayed as requested.
