Add/Edit Crash Report

The Add Crash Report screens allows you to add a crash report. This is useful in the case where a crash occurs in a vehicle that is not equipped with a tiwiPro device, or one that has a device that does not have crash detection enabled.  The crash must be assigned to a specific driver and vehicle.  You must also specify the time the crash occurred.  The crash location can be specified either by associating it with a trip or by selecting 'Find Address' and selecting a location by clicking on the map.

You can also add weather details, occupant count and a description.

Click on Save to save any information that has been changed.  Click on Cancel to revert your changes.

 

Crash Details Edit screens allows you to edit certain details about a selected crash or roll over.   If the crash was an actual crash that was detected by the device,  the date and crash location cannot be modified.

You can edit the crash status, weather details, occupant count and add a description.

Click on Save to save any information that has been changed.  Click on Cancel to revert your changes.
