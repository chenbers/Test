Add/Edit/Batch Edit Vehicles - Details

To batch edit vehicle details, click here.

On this page, you can add or edit vehicle details. There are two tabs you need to fill out: the Details tab, and the Speed & Sensitivity tab. The Details tab is shown below.

**NOTE**

Please note that the Save button is only found on the Details page. You will need to set the Speed and Sensitivity settings prior to clicking Save. Otherwise, you'll need to select the vehicle in from the Vehicles tab, then click Edit to change the settings.

To learn more about various areas of interest, move your mouse over the image below. When you see the icon, click that area. A pop-up window will appear.

Batch Edit Vehicle Details

In the instances where you would like to batch edit multiple vehicles, check the boxes next to the fields that you wish to batch edit. Only those fields will be updated with the batch edited content.
