FusionCharts ASP Class API

FusionCharts XT can be integrated with ASP to produce interactive, data-driven charts. Using with ASP section documents charting data from ASP array and forms, connecting to database to fetch data and create snazzy charts out of them and many more sample applications.

While developing these applications in ASP, we realized that a set of built-in ASP functions could save us a lot of time by performing various chart generation tasks. Having that said, we release FusionCharts ASP Class for FusionCharts XT! This class has some ASP functions that can be used while configuring and rendering the chart, and for XML generation.

In the coming sections we will see how to use FusionCharts ASP Class effectively so that charting with ASP becomes easier than ever!
