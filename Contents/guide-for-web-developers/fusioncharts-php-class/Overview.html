FusionCharts PHP Class API

FusionCharts XT can be integrated with PHP to output interactive, data-driven charts. The Using with PHP section documents charting data from PHP array and forms, connecting to database to fetch data and creating snazzy charts out of them and many more sample applications.

While developing these applications in PHP, we realized that a set of built-in PHP functions could save us a lot of time by performing various chart generation tasks! Having said that, we released the FusionCharts PHP Class for FusionCharts XT! This class has some PHP functions that can be used while chart configuration , XML generation and rendering.

In the coming sections we will see how to use FusionCharts PHP Class effectively so that charting with PHP becomes easier than ever!
